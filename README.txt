aplikasi ini dibuat dengan DaisyUI(Tailwind css), ExpressJS, EJS, mongoose(MongoDB), dan masih banyak package lainnya...

sebelum di run pastikan dulu sudah update dependecies nya
dengan mengetikan ini di terminal : 
"npm install"

jangan lupa pastikan sudah terinstall MongoDB pada perangkat anda

Lalu anda dapat run program dengan mengetikan ini di terminal :
"npm run devStart"

maka aplikasi akan ter run di env.PORT atau di port 5000


dan perlu catatan juga untuk tampilannya memungkinkan untuk berbeda di setiap device, dikarenakan penyettingan display yang berbeda,
pada project ini, kita memakai display setting pada windows nya di scale 100%.
